Персональный сайт-портфолио, представляющий информацию о моих навыках, стеке технологий и выполненных проектах. Сайт выполнен в минималистичном стиле с поддержкой темной темы.

## 🚀 Особенности проекта

* **Адаптивный дизайн:** Корректное отображение на мобильных устройствах и десктопах.
* **Темная тема:** Реализовано переключение цветовой схемы (Dark/Light Mode).
* **Сохранение состояния:** Выбранная тема сохраняется в `localStorage` браузера, поэтому пользователю не нужно переключать её заново при обновлении страницы или переходе по ссылкам.
* **Интерактивные элементы:** Анимированные меню с использованием CSS-псевдоэлементов.

## 🛠 Стек технологий

* **HTML5:** Семантическая разметка страниц.
* **CSS3:** Использование CSS-переменных (`--variables`), Grid Layout и Flexbox для построения сеток.
* **JavaScript (ES6+):** Логика переключения тем и работа с локальным хранилищем.

## 📂 Структура проекта

* `index.html` — Главная страница с информацией "О себе" и стеком технологий.
* `projects.html` — Список реализованных проектов.
* `details.html` — Подробное описание конкретного проекта (на примере Менеджера паролей).
* `style.css` — Стилизация всего сайта, включая анимации и переменные для тем.
* `scripts.js` — Скрипты для интерактивности и управления состоянием темы.
